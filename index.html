pExporter

This extension allow you to download playlists from Deezer.com You can save the file as CSV or Goovylist (TXT) format to easily import them elsewhere.

Warning: This extension doesn't allow you to download music from Deezer.com

If you want explanations about how it works, simply visit http://dblk.org/pexporter or download it from Google Web Store
